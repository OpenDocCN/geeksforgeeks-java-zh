# 求数列 1，8，54，384 的第 n 项……

> 原文:[https://www . geesforgeks . org/find-n 系列术语-1-8-54-384/](https://www.geeksforgeeks.org/find-nth-term-of-the-series-1-8-54-384/)

给定一个数字 n，任务是编写一个程序来寻找下面系列中的第 n 项:

```java
1, 8, 54, 384...
```

**例:**

```java
Input : 3
Output : 54
For N = 3
Nth term = ( 3*3) * 3!
         = 54

Input : 2 
Output : 8
```

仔细观察，上述级数中的第 n 项可以概括为:

```java
Nth term = ( N*N ) * ( N! )
```

以下是上述方法的实现:

## C++

```java
// CPP program to find N-th term of the series:
// 1, 8, 54, 384...
#include <iostream>
using namespace std;

// calculate factorial of N
int fact(int N)
{
    int i, product = 1;
    for (i = 1; i <= N; i++)
        product = product * i;
    return product;
}

// calculate Nth term of series
int nthTerm(int N)
{
    return (N * N) * fact(N);
}

// Driver Function
int main()
{
    int N = 4;

    cout << nthTerm(N);

    return 0;
}
```
